SOURCE CODE (2 new files)
================================================================================

1. src/sliding_window_chunker.py
   - SlidingWindowChunker class for creating overlapping chunks
   - Two chunking strategies:
     * chunk_with_sliding_window() - Pure token-based sliding window
     * chunk_by_smart_boundaries() - Respects sentence/paragraph boundaries
   - WindowedChunk dataclass with overlap metadata
   - Default: 1024 tokens/chunk, 30% overlap

2. src/note_merger.py
   - NoteMerger class for intelligent note deduplication
   - Two merging strategies:
     * merge_consecutive_notes() - Merge pairs with Claude
     * merge_notes_in_batches() - Hierarchical batch merging
     * merge_all_to_single() - Merge all notes into one comprehensive note
   - Uses Claude API to eliminate redundancy while preserving content
